# ELECTROVERSE CONNECT

Welcome to the **ELECTROVERSE** academic portal for the B.Tech Electrical Engineering Department at Nalanda Institute of Technology, Bhubaneswar.

## Tech Stack
- **Framework:** Next.js 15 (App Router)
- **Styling:** Tailwind CSS + Shadcn/UI
- **Backend:** Firebase (Authentication & Firestore)
- **AI:** Genkit (Electrical Concept Explainer & Study Resource Generator)

## Getting Started

To push this project to your GitHub repository, run the following commands in your terminal:

```bash
git init
git remote add origin https://github.com/Hitakankshi/Electroverseconnect.git
git add .
git commit -m "Initial commit: ELECTROVERSE Connect Portal"
git push -u origin main
```

## Features
- **Student Portal:** Access CGPA, attendance, and academic profiles.
- **Admin Dashboard:** Manage student records and media assets.
- **AI Tutor:** Interactive tool for explaining complex electrical engineering concepts.
- **Gallery:** Visual showcase of achievements and campus events.
- **Results:** Public portal for checking examination summaries.
